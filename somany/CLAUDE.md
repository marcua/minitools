# So Many! - Counting Games PWA

Single-file PWA with toddler-friendly counting games using neobrutalism design.

See [README.md](./README.md) for full developer guide including:
- Architecture and game structure
- Design system (neobrutalism)
- Adding new games
- Common gotchas

## Critical Reminder
**Always bump `CACHE_NAME` in `sw.js` after any change to `index.html`**
or users won't see updates.
